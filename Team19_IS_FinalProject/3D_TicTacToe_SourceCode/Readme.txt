3D 4x4 Tic-Tac-Toe Game
Overview:

This repository features a 3D 4x4 Tic-Tac-Toe game developed in Java. The primary file, Tic_Tac_Toe_3D.java, is compatible with Visual Studio Code, enabling an engaging twist on the classic game with enhanced gameplay mechanics.

How to Run:

Install Visual Studio Code and open the Extensions view (Ctrl+Shift+X). Search for and install the Java Extension Pack, which includes all necessary tools for Java development.
Open the project folder in Visual Studio Code.
If prompted, install and configure the required Java extensions.
Locate the Tic_Tac_Toe_3D.java file.
Right-click inside the editor and choose Run Java to start the application.
Game Description:

This game reimagines Tic-Tac-Toe in a 3D space using a 4x4x4 grid. Players alternate placing their symbols (X or O), aiming to form a winning line across rows, columns, or diagonals in three dimensions, adding a strategic challenge to the traditional game.
